1. What is the difference between static and relative positioning?

r: Static is the default positioning of an element and always follows the same direction (down) and relative

2. What is absolute positioning useful for?

r: To remove an element of the actual flow. Don't affect other elements where it is placed.
(Good for icons, images with captions, etc.)

3. What is the difference between fixed and sticky positioning?

r: Fixed is always at the same position unlike sticky that it can be placed where we want but being scroll down make it
fixed to the top of the page.
